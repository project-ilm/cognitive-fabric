# Context

This repository originated from investigating the practical
deployment of fully local FOSS talking-face systems on a
Ryzen 7 + RTX3050 (4GB VRAM) laptop.

The investigation gradually expanded into a much broader
research question:

How should conversational digital humans be architected,
verified, deployed and benchmarked across every deployment
profile from edge devices to research clusters?

The repository therefore shifted from an application
repository into an architectural reference implementation.
